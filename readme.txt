Developed a FoodShala web application which enables the users to order a food and restaurants to provide foods online.
To use this project one will need a web server to run php code and a mysql database.
To setup a database database name should be foodshala, username should be your databse usersname or root, and password should database password.
Then one can replicate the database using foodshala.sql.
And can use foodshala web application using url http://127.0.0.1/foodshala
